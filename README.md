# medical-data-agent
A local-first medical data analysis agent prototype with automated data mapping ADaM-like dataset generation TLF production CSR-style reporting and audit-ready traceability


log01:

I built the first prototype of a local medical data analysis pipeline.
It reads raw client-style clinical data
profiles the metadata
automatically generates a draft mapping from raw columns to ADaM-like variables
saves the mapping configuration for auditability
builds an analysis-ready ADSL-like dataset
generates a demographics and baseline characteristics table
and produces a mini CSR-style report with traceability notes
